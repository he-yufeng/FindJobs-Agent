## 🚀 快速开始（给评审老师）

### 方式1: 使用一键启动脚本（推荐）

**Windows:**
```bash
cd code_submission
scripts\start_windows.bat
```

**Linux/macOS:**
```bash
cd code_submission
chmod +x scripts/start_linux.sh
./scripts/start_linux.sh
```

### 方式2: 手动启动

1. **安装依赖**
```bash
cd code_submission
pip install -r requirements.txt
```

2. **配置API密钥** (可选，用于完整功能)
```bash
cp API_key-openai.md.template API_key-openai.md
# 编辑 API_key-openai.md 添加你的OpenAI API密钥
```

3. **启动后端** (终端1)
```bash
python api_server.py
```

4. **启动前端** (终端2)
```bash
cd FrontEnd
npm install
npm run dev
```

5. **访问应用**
- 前端: http://localhost:5173
- 后端: http://localhost:5000
